GIT / BASH COMMANDS

GIT

  • git init:Initializes repo for local project folder
  • git config: --global user.name "" / --global user.email ""
  • git add. : Adds all untracked changes or just a file
  • git commit -m: Records changes to repo with a message
  • git push origin main: Uploads local repo content to remote repo
  • git pull: Fetches and merges changes from a remote repo to your local repo
  • git status:Displays status of the working directory.
  • git stash:Lets you switch branches or do other things wihout discarding or commiting changes that aren't ready.
  • git --amend: Combine staged changes with the previous commit instead of creating a new commit.
  • git branch: New separate version of the main repo.
  • git log:Shows commit history of repo.
  • git merge:Integrates changes from one branch to another.
  • git switch: switches to specified directory.
  • git switch -c:creates and switches to the specified branch

BASH

  • pwd: Prints working directory.
  • mkdir: Creates a directory.
  • cd:Change directory.
  • touch:Creates a new file.
  • ls:Lists directory contents.
  • cp (filename directory-name):Copies a file to a new directory.
  • cat (options file path):Displays the contents files without opening them to edit.
  • mv(options source destination): Moves directories and files within the same file system.
  • grep:Searches for patterns.
  • echo:Displays a line of text or string that is passed as an argument.
  • rm:Removes files or directories.
